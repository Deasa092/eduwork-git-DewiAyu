Buatlah sebuah aplikasi dengan ketentuan sebagai berikut:

  1. Buatlah 3 buah variable
  2. Variable pertama berupa angka 10 dan variable kedua berupa tulisan “saya belajar javascript”
  3. Lalu jumlahkan antara variable pertama dan kedua
  4. Apabila anda menemukan keanehan atau masalah maka silahkan jumlahkan dengan angka berapapun menggunakan variable ketiga untuk menggantikan variable yang bermasalah
